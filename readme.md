git init 初始化仓库
git status 查看仓库中文件的状态，untracked（未跟踪，没上传到git），unmodify（修改）， modified（修改）， staged（暂存区）
git status -s （简写）
git add xx.html 跟踪文件
git commit -m "提交添加文字描述"
